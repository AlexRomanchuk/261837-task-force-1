Задание

Класс Задание

Содержит статус задания и изменяет его. Получает данные о задании из базы данных.

<--

Класс Валидатор действия пользователя

Содержит валидацию действий пользователя. Получает данные о пользователе. Определяет права пользователя на изменение статуса и запрещает или разрешает действие.

Класс Отклик на задание

Содержит отклик на задание, статус отклика, данные о исполнители. Может быть отклонен или подтвержден. При добавлении отправляется уведомление. При подтверждении изменяет статус задания, добавляя в него исполнителя.

Класс Завершение

Содержит данные о задании и заказчике. Может содержать ответ Завершен или Есть проблемы. Изменяет статус задания.

Пользователь

Класс Пользователь

Содержит данные о пользователе. Получает данные из базы данных. В аккаунте может изменить свой статус и данные. Права зависят от роли (заказчик или исполнитель). Связан с валидатором действий.

------------------^

Класс Отзыв о работе пользователя

Содержит данные о пользователе, фото и текст отзыва. Сохраняется в базе данных.

Класс Уведомление

Содержит текст уведомления. Сохраняется в базе данных. Может быть прочитано.
